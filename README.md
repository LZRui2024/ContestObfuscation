# ContestObfuscation

### ⚠警告⚠: 建议您不要在编程科上考试上使用该工具，否则可能会被老师创飞

目前适配的C++语句不算太多，所以可能会出现编译错误

用于直接混淆CPP (编程竞赛类) 源码的工具 (目前属于测试阶段, 仅供参考)

目前工具拥有混淆功能:

  1.函数名混淆
  2.宏混淆(通过宏增加阅读难度)
  
  3.注入无关代码
  
  4.添加无用函数
  
  5.添加函数跳转
  
  6.变量名混淆
  
  7.常量加密
  
  8.字符串加密
  
  9.代码分块
  
  10.指令替换
  
  11.字符串拆分与重组
  
  12.函数内联与反内联
  
  13.循环变换
  
  14.异常处理混淆
